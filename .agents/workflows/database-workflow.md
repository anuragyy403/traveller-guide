# Database Workflow

When this workflow is triggered:

1. Follow MongoDB best practices.
2. Use Mongoose best practices.
3. Design efficient schemas.
4. Minimize unnecessary queries.
5. Recommend indexes when needed.
6. Validate inputs.
7. Preserve existing data structures.
8. Consider scalability.
9. Consider security implications.
10. Explain migration requirements if needed.

Response Format:

SCHEMA:
<schema recommendation>

QUERY:
<query recommendation>

OPTIMIZATION:
<performance improvements>

NOTES:
<additional considerations>
